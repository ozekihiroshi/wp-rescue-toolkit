# WordPress Maintenance Work Checklist

## 1. Before Work

- [ ] Client confirmed work scope
- [ ] Site URL confirmed
- [ ] Admin login available
- [ ] Hosting/cPanel/SSH access available if needed
- [ ] Maintenance window confirmed
- [ ] Backup method confirmed
- [ ] Rollback method confirmed

## 2. Current Environment Check

- [ ] WordPress version checked
- [ ] PHP version checked
- [ ] MySQL/MariaDB version checked
- [ ] Active theme checked
- [ ] Child theme checked
- [ ] Active plugins listed
- [ ] Critical plugins identified
- [ ] Cache/CDN status checked
- [ ] Security plugin status checked

## 3. Backup Check

- [ ] Database backup exists
- [ ] WordPress files backup exists
- [ ] wp-config.php included
- [ ] uploads directory included
- [ ] themes directory included
- [ ] plugins directory included
- [ ] Backup location confirmed
- [ ] Restore method understood

## 4. Update Work

- [ ] Plugin updates reviewed
- [ ] Theme updates reviewed
- [ ] WordPress core update reviewed
- [ ] PHP compatibility checked
- [ ] Staging tested if available
- [ ] High-risk updates separated
- [ ] Updates applied carefully
- [ ] Error logs checked after update

## 5. Post-Update Validation

- [ ] Homepage checked
- [ ] Key pages checked
- [ ] Login checked
- [ ] Contact forms checked
- [ ] Checkout/payment checked if applicable
- [ ] Admin dashboard checked
- [ ] PHP errors checked
- [ ] Browser console checked if needed
- [ ] Cache purged if needed
- [ ] CDN behavior checked if needed

## 6. Report

- [ ] Work performed summarized
- [ ] Issues found summarized
- [ ] Actions taken summarized
- [ ] Remaining risks listed
- [ ] Recommended next steps listed
